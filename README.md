# Next.js Admin Dashboard (TailAdmin Style)

This project is a sample admin dashboard template, inspired by the [TailAdmin Tailwind CSS Admin Dashboard Template](https://demo.tailadmin.com/). It is rebuilt using **Next.js**, **Tailwind CSS**, and **daisyUI**.

This project is **totally free and open-source**.

## ğŸ Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd nextjs-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.
